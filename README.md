# kotlin-baseball-precourse

## 기능 목록
- 함수(또는 메서드)가 한 가지 일만 하도록 최대한 작게 만들기

### 클래스

- 게임 매니저
    - new 게임 시작하기
        - 컴퓨터의 숫자 정하기
            - 1~9까지 서로 다른 임의의 수 3개
        - 숫자 입력하기
            - 잘못된 값을 입력하면 `IllegalArgumentException` 발생하고 종료
        - 나의 숫자 채점하기 
          - 스트라이크:같은 수가 같은 자리에 있을 때 
          - 볼:같은 수가 다른 자리에 있을 때 
          - 낫싱:같은 수가 전혀 없을 때
    - re game 여부 물어보기
